01222000 -- Jan 22, 2000

  • putXML
    • The XML elements don't have to match the relative ordering of their corresponding db. columns or ADT attributes.
    • Expose  insert batch size and the commit batch size for fine performance tuning
    • Significant performance improvements
  • getXML
    • API (java & pl/sql) support for name binds in the SQL query statement
    • API (Java & PL/SQL) support for header insertion into the DTD and the XML document
    • API (Java & PL/SQL) support for generating the XML meta doc only (i.e. gen. the DTD only)
    • API (Java & PL/SQL) support for propagating the original exception
    • support for cursors in the SQL query
    • added support for specifying the desired characterset for the generated doc
    • Performance improvements
  • common
    • created a JDK1.2/JDBC2.0 compliant version of the XSU (i.e. XSU12)
    • misc bug fixes
    • the xmlgen PL/SQL package is set to execute with invokers-rights

  • Added BFILE support in generation
  • Changed allocation duration to CALL for temp lobs in OracleXMLStore
